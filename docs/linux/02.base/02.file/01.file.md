## chown
* 소유자 설정
```bash
chown -Rc (소유자):(그룹) (파일명)
```
* `-R`: 재귀적으로 하위 디렉토리까지 적용.
* `-c`: 변경된 파일 대상 verbose 모드. 작업 상태 출력.
